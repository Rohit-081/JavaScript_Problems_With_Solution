Requirement

Given two non-negative integers, num1 and num2 represented as string, return the sum of num1 and num2 as a string.

Example 1: Input: num1 = "11", num2 = "123" Output: "134"

Example 2: Input: num1 = "456", num2 = "77" Output: "533"

Example 3: Input: num1 = "0", num2 = "0" Output: "0"

Constraints:

1 less than equal to num1.length, num2.length less than equal to 10^4

num1 and num2 consist of only digits.

num1 and num2 don 't have any leading zeros except for the zero itself.

Pseudo Code:

1. Given two non-negative integers, num1 and num2 represented as string

2. Firstly I will find out which is greater in length and swap it.

3. Traverse in such a way so that i will add no.s from right to left using a carry variable.

4. That carry variable is used to store the value of sum of the two variable after getting any carry value i will assign it to the next left value which we gonna do add.

5. This is going from right to left and we will print the output.
